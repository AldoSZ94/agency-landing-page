# ⭐ Agency Landing Page

Proyecto de maquetación web basado en el reto de Frontend Mentor **"Agency Landing Page"**.  
El objetivo fue construir una landing page moderna, completamente responsive, aplicando diseño mobile-first, layout avanzado con CSS Grid y Flexbox, e interacción dinámica en el menú de navegación mediante JavaScript vanilla, integrando manejo de estado y buenas prácticas de accesibilidad web.

---

## 📌 Descripción

Este proyecto consiste en una landing page para una agencia creativa con secciones informativas y visuales que destacan servicios, testimonios y galería de imágenes.

Incluye un encabezado principal con navegación responsive, un hero section con imagen de fondo y varias secciones de contenido diseñadas para mantener una jerarquía visual clara y una experiencia de usuario fluida.

Se puso especial énfasis en:

- Estructura semántica HTML  
- Diseño responsive con enfoque mobile-first  
- Layout moderno con CSS Grid y Flexbox  
- Interactividad con JavaScript vanilla  
- Manejo de estado del menú de navegación  
- Accesibilidad web mediante atributos ARIA  
- Experiencia de usuario (UX) con transiciones suaves  
- Código limpio, organizado y escalable

## 🎯 Características principales

- 📱 Diseño completamente responsive para mobile, tablet y desktop  
- 🍔 Menú hamburguesa interactivo con estado controlado en JavaScript  
- 🎨 Hero section con imagen de fondo adaptable (mobile/desktop)   
- ⚡ Transiciones suaves usando utilidades de Tailwind CSS  
- 🖼️ Uso de `<picture>` para imágenes responsivas  
- ♿ Implementación de accesibilidad con `aria-expanded` y `aria-controls`  
- 🧱 Layout complejo con CSS Grid y Flexbox  
- 🎯 Secciones estructuradas: servicios, testimonios y galería

## 🧠 Aprendizaje

Durante este proyecto reforcé conceptos clave como:

- Manipulación del DOM con JavaScript  
- Manejo de eventos con `addEventListener()`  
- Control de estado del menú de navegación  
- Uso de `classList.toggle()` para UI dinámica  
- Implementación de navegación responsive tipo dropdown  
- Mejora de accesibilidad con atributos ARIA  
- Uso de `<picture>` para imágenes responsivas  
- Construcción de layouts complejos con CSS Grid  
- Aplicación de diseño mobile-first  
- Separación de estructura, estilo e interacción
